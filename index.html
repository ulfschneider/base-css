Notation

Throughout this document the following notation is applied:

div /div HTML tags are indicated by their name. The closing HTML tag is indicated with a prefixed slash.
div/ The abbreviated form of an opening and closing HTML tag is indicated by the tag name with a trailing slash.
.left CSS classes are indicated with a prefixed dot
.left.middle The assignment of multiple CSS classes to a single HTML tag, composing, is indicated with a prefixed dot for each CSS class; and there is no white space between classes.
i, .i, .italic Multiple styling options that produce the same result are indicated by HTML tag names or CSS class names, each one seperated by comma.

Apply Colors to Text, Background, and Borders

Set color, background-color and border-color by applying the naming conventions:

Color Background Border Refine
.neutral .bg-neutral .br-neutral neutral-d4 … neutral-4
.primary .bg-primary .br-primary primary-d4 … primary-4
.accent-one .bg-accent-one .br-accent-one accent-one-d4 … accent-one-4
.accent-two .bg-accent-two .br-accent-two accent-two-d4 … accent-two-4
.accent-three .bg-accent-three .br-accent-three accent-three-d4 … accent-three-4

Edge Values

In addition there are four colors available, which can also be used as color, background-color and border-color but do not allow further refinement:

  • .base-color (which is default text-color).

    The color is defined through:

    :root { --base-color = var(--neutral-color-d4); }
  • .base-background

    The color is defined through:

    :root { --base-background = #fff; }
  • .black

    The color is defined through:

    :root { --black = black; }
  • .white

    The color is defined through:

    :root { --white = white; }
  • .mark, mark

    The color is defined through:

    :root { --mark-background: var(--accent-three-2); }
  • .kbd, kbd

    The color is defined through:

    :root { --kbd-background: var(--neutral-4); }

Define Your Own Color System

The color system is built around the five colors:

  • neutral
  • primary
  • accent one
  • accent two
  • accent three

To define your own color palette, change the colors by providing their hsl color code.

:root {
    --neutral-h: 0;
    --neutral-s: 0%;
    --neutral-l: 53%;              
    
    --primary-h: 213;
    --primary-s: 19%;
    --primary-l: 60%;
    
    --accent-one-h: 3;
    --accent-one-s: 100%;
    --accent-one-l: 61%;  

    --accent-two-h: 28;
    --accent-two-s: 100%;
    --accent-two-l: 55%;  
    
    --accent-three-h: 52;
    --accent-three-s: 100%;
    --accent-three-l: 50%;  
}

Images

Responsive Images

Images are by default responsive and take the entire available horizontal space while preserving aspect-ratio. 

img/

Contained Images

If the image has a fixed width or fixed height, the usage of .fix-contain will put the entire visible image inside of the given dimensions while preserving aspect-ratio and without stretching the image.

Combine .fit-contain with fit positions .fit-t, .fit-tl, .fit-tr, .fit-r, .fit-b, .fit-bl, .fit-br, and .fit-l.

img/ .fit-contain
img/ .fit-contain.fit-tl

Cover Images

If the image has a fixed widht or fixed height, the usage of .fix-cover will consume the entire available space of the given dimensions while preserving aspect-ratio.

The image might be stretched to cover.

img/ .fit-cover

Background Images

Background images can also be positioned with .fit-cover, which is usefule for hero banners.

Hero
div/ .fit-cover style="height: 600px; background-image: url('image.jpg')"

12 Column Grid

Any space can be divided horizontally into 12ths.

The sum of the cols per row has to add up to 12. Not specifiying a column size will result in taking the entire grid width into one column.

It is possible to have differently sized columns based on minimum screen width. Address the different minimum screen sizes with column prefix values of lg- (for at least large size), md- (for at least medium size), rg- (for at least regular size) and sm- (for at least small size). A column class without a prefix will be used in case the alternate minimum screen size is not present.
