Grid

.x

Middle content

.x *

Middle content is the default positioning of any item within the layout. Middle content doesn´t need a specific css class assignment to get positioned this way.

Code example:

<div class="x">
<div></div>
</div>

Breakout left content

.x .breakout-l

Breakout left content is pushed a little bit to the left and will give more space for the things you want to display.

Code example:

<div class="x">
<div class="breakout-l"></div>
</div>

Breakout right content

.x .breakout-r

Breakout right content is pushed a little bit to the right and will give more space for the things you want to display.

Code example:

<div class="x">
<div class="breakout-r"></div>
</div>

Breakout content = left and right

.x .breakout

Breakout content is the combination of left and right content.

Code example:

<div class="x">
<div class="breakout"></div>
</div>

Wide content

.x .wide

Wide content is pushing things further in that it nearly takes the full width of the screen, except of a small margin to the window border.

Code example:

<div class="x">
<div class="wide"></div>
</div>

Full content

.x .full

Full content will use the entire width of the screen.

Code example:

<div class="x">
<div class="full"></div>
</div>

12 column grid

.x-12 .col-*

Any of the before mentioned content concepts can be divided horizontally into 12ths.

The sum of the cols per row have to add up to 12. Not specifiying a column size will result in taking the entire grid width into one column.

Code example:

<div class="x-12">
<div class="col-12"></div>
<div class="col-4"></div>
<div class="col-4"></div>
<div class="col-4"></div>
<div class="col-6"></div>
<div class="col-6"></div>
<div class="col-2"></div>
<div class="col-10"></div>
</div>

It is also possible to have differently sized columns based on minimum screen width. Address the different minimum screen sizes with column prefix values of lg- (for at least large size), md- (for at least medium size), rg- (for at least regular size) and sm- (for at least small size). A column class without a prefix will be used in case the alternate minimum screen size is not present.

Code example:

<div class="x-12">
<div"></div>
<div class="lg-col-2 md-col-4 col-6"></div>
<div class="lg-col-2 md-col-4 col-6"></div>
<div class="lg-col-2 md-col-4 col-12></div>
<div class="col-6"></div>
<div class="col-6"></div>
<div class="md-col-3"></div>
<div class="md-col-9"></div>
</div>

Columns

.columns

Text will be displayed in columns. The number of columns depends on the available width.

<div class="x">
<div class="breakout-r columns"></div>
</div>
