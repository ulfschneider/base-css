A CSS GRID SYSTEM TO LAYOUT A WEB PAGE

The Grid

A system to layout
April 2018
by Ulf Schneider
This site is using the CSS Grid to provide a fully responsive layout. Though not supported by IE, the grid system is robust, lightweight and fast. This is of advantage for users as well as for the developer. Grid is the future of page layout.

Middle Content

Middle content is the default positioning of any item within the layout. Middle content doesn´t need a specific cssl class assignment to get positioned this way.

<div class="grid">
<div></div>
</div>

Left Content

Left content is pushed a little bit to the left and will give more space for the things you want to display. Left content needs to have a css class named left assigned.

<div class="grid">
<div class="left"></div>
</div>

Right Content

Right content is pushed a little bit to the right and will give more space for the things you want to display. Right content needs to have a css class named right assigned.

<div class="grid">
<div class="right"></div>
</div>

Breakout Content

Breakout content is the combination of left and right content. Breakout content needs to have a css class named breakout assigned.

<div class="grid">
<div class="breakout"></div>
</div>

Wide Content

Wide content is pushing things further in that it extends the boundaries of breakout content. Wide content needs to have a css class named wide assigned.

<div class="grid">
<div class="wide"></div>
</div>

Full Content

Full content will use the entire width of the screen. Full content needs to have a css class named full assigned.

<div class="grid">
<div class="full"></div>
</div>

Thirds

Any of the before mentioned content concepts can be divided into thirds by assigning the css class named thirds.

<div class="grid">
<div class="wide thirds">
<div class="col-1-2"></div>
<div class="col-3"></div>
</div>
</div>

Below the concept is applied to position two images.

Columns

Text will be displayed in columns, when a class named columns is assigned to the text. The number of columns depends on the available width.
