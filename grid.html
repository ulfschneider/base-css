A HORIZONTAL CSS GRID SYSTEM TO LAYOUT A WEB PAGE

This site is using the CSS Grid to provide a fully responsive layout. Though not supported by IE, the grid system is robust, lightweight and fast. This is of advantage for users as well as for the developer. Grid is the future of page layout.

If display: grid; is not supported by the web browser, everything is positioned in one column in the middle of the page.

Content

Middle content is the default positioning of any item within the layout. Middle content doesn´t need a specific css class assignment to get positioned this way.

Code example:

<div class="grid">
<div></div>
</div>

Breakout Left Content

Breakout left content is pushed a little bit to the left and will give more space for the things you want to display. Breakout left content needs to have a css class named breakout-left assigned.

Code example:

<div class="grid">
<div class="breakout-left"></div>
</div>

Breakout Right Content

Breakout right content is pushed a little bit to the right and will give more space for the things you want to display. Breakout right content needs to have a css class named breakout-right assigned.

Code example:

<div class="grid">
<div class="breakout-right"></div>
</div>

Breakout Content = Left and Right

Breakout content is the combination of left and right content. Breakout content needs to have a css class named breakout assigned.

Code example:

<div class="grid">
<div class="breakout"></div>
</div>

Wide Content

Wide content is pushing things further in that it nearly takes the full width of the screen, except of a small margin to the window border. Wide content needs to have a css class named wide assigned.

Code example:

<div class="grid">
<div class="wide"></div>
</div>

Full Content

Full content will use the entire width of the screen. Full content needs to have a css class named full assigned.

Code example:

<div class="grid">
<div class="full"></div>
</div>

Sixths

Any of the before mentioned content concepts can be divided into sixths by assigning the css class named six.

Code example:

<div class="grid">
<div class="wide six">
<div class="six-1-to-4"></div>
<div class="six-3-to-5"></div>
<div class="six-1"></div>
<div class="six-2-to-3"></div>
<div class="six-1-to-2"></div>
<div class="six-3-to-4"></div>
<div class="six-5-to-6"></div>
<div class="six-1"></div>
<div class="six-2"></div>
<div class="six-3"></div>
<div class="six-4"></div>
<div class="six-5"></div>
<div class="six-6"></div>
<div class="six-1-to-3"></div>
<div class="six-4-to-6"></div>
</div>
</div>

Columns

Text will be displayed in columns, when a class named columns is assigned to the container surrounding the text. The number of columns depends on the available width.
